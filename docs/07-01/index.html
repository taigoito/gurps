世界観

魔物

物語の舞台

GURPS.QWELの舞台設定は、術法と同じくして某有名RPGから抜擢させていただいています。地域の名称もそのまま使わせていただきますが、ゲームの攻略サイトとは異なりますので注意してください。

その世界は、私たちの世界とは異なる自然法則で成り立っていて、万物には不可視な霊魂（のようなもの）が宿っていると考えられ、それに一定の指向性を与え、何か具体的な現象（術）を引き起こす技術を、「術法」と呼んでいます。また、人々の自然に対する畏怖は、どこかで秘かに具現化され、魔物を発生させています。

「術法」の存在は、私たちの世界の「科学」のような文明を形成するほど洗練されたものではなく、かつ生活の身近に普及しているものでもありません。また魔物の存在は、生態系に組み込まれた野生動物の延長ではなく、神や霊に近い畏怖の対象です。そういった意味では、私たちの世界と何ら変わりないとも言えます。

人間社会の文化・文明は、中世ヨーロッパの時代を踏襲します。つまり産業革命・大航海時代の以前であり、文明は未熟で、世界の全貌も分かっていません。

しかし実は、超古代文明が存在し、古代人達は、動物や植物、さらには魔物の魂を自らと同化させ、不老不死の力を得ていました。そのような生命の摂理に反する行為が、魔物を大量発生させてしまい、文明は滅んでしまったのです。

物語の主人公たちは、超古代文明の頃から千年余り経過し、魔物たちが人々の暮らしを脅かしている時代を冒険します。

ゴブリン

魔物は、人々の自然に対する畏怖が具現化されて、発生します。

最もポピュラーな魔物である「ゴブリン」は、母親が子供をしつけるときに、「悪い子だとゴブリンに連れていかれてしまうよ！」などと言ったときに、子供の怖れが具現化し、どこか近くの森にゴブリンを産み落とすのです。

ゴブリンは自ら繁殖し、勢力を拡大することもありますが、さほどの脅威ではありません。普段は狩猟採集の生活をしていて、時々、人間の集落から家畜や子供をさらっていく程度です。家畜は食べてしまうでしょうが、子供は無垢で怖れを知らなければ、ゴブリンが子供を殺してしまうことはありません。

光り物が好きで、ゴブリンの巣窟には、金銀財宝が蓄えられていることもあります。

ゴブリンの強さは、10歳位の人間の子供と同程度です。剣や棍棒を振るいますが、非力なので、攻撃が当たっても致命傷には至らないでしょう。能力値は下記表を参考にしてください。

人間の大人並みの体格をした「ホブリン」もいます。

ゲットー

ウサギを連想させる長い耳と、俊敏な足、そして毛皮に覆われた亜人を「ゲットー」と呼びます。ゴブリンよりも華奢で賢く、弓を引いたり、罠を作ったりすることができます。耳が良いため、ゴブリンの巣窟で家来のように扱われ、見張りとして立たされていることがありますが、基本的には臆病者で、単体で遭遇しても逃げ出してしまうことでしょう。

グール

未練を残して死亡した人の思念が具現化されると、「グール」と呼ばれる死肉を食い漁る魔物を発生させることがあります。ゾンビのような形態をしていて、鋭い爪や牙で襲い掛かってきます。

グールは特性上、「刺し」や「切り」の武器で攻撃しても、ダメージ増加がありません。HP0になると、破壊されて活動を停止します。

グールは洞窟や古戦場などで遭遇することがあるかも知れません。ただし日の光に弱く、日中は屋外で活動できません。

スカル・ウォーリア / スカル・メイジ

戦場で散った兵士の怨霊が、白骨化した死体や武具に宿り、生者を見境なく襲ってくる魔物と化すことがあります。それが剣士であればスカル・ウォーリア、術士であればスカル・メイジと呼ばれます。生前の剣術や術法を駆使して戦います。

グールと同じ特性を持ち、「刺し」や「切り」の武器で攻撃しても、ダメージ増加がありません。また日の光に弱く、日中は屋外で活動できません。

スライム

ゼリー状の透明な物質で満たされ、中心に核を持つ単細胞の形態をした魔物です。弾力のある身体で、飛び跳ねて打撃したり、溶解液を吐出したりして攻撃します。

スライムは特性上、「刺し」や「切り」の武器で攻撃したときのダメージ増加分でしかダメージを与えることができません。

スライムはどのように発生するのかは不明ですが、特定の条件で大量に出現することもあります。分裂して増殖するという説もありますが、定かではありません。

魔獣 / 聖獣

巨大蜘蛛や、三頭を持つ犬など、動物や植物の形態に近い魔物は、「魔獣」と呼びます。またユニコーンやグリフィンなど、人々の崇拝の対象となり得る「聖獣」も存在します。

やはりどのように発生するかは不明ですが、ほとんどは単独で行動し、繁殖することはありません。野山を探索していても、偶然に遭遇することはほとんどありません。

異形

牛頭の巨人「ミノタウロス」や、半人半馬「ケンタウロス」など、古代人が異生物と同化して成り果てた姿の者を「異形」と呼びます。どれだけ人間の知性が残っているかはそれぞれですが、手強い敵となるのは明らかです。遺跡などで遭遇することもあるかも知れませんが、ほとんど伝説上の存在であり、冒険者も滅多に出会うことはありません。
