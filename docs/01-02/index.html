キャラクター

CPと能力値

CP(キャラクターポイント)の振り分け

キャラクター作成は、CP(キャラクター・ポイント)の持ち点を10点とし、これを「能力値」、「技能」、「特徴」に振り分けることから始まります。

持ち点が足りないと感じたら、不利な能力や特徴を決め、持ち点を増やすこともできます。能力値の下降は、初期値-2までとします。

能力値一覧
能力値 説明
ST(腕力) 力の強さ。攻撃の命中、敵に与えるダメージ、HPに関係します。10を初期値とし、2CPより上昇・下降できます。
DX(器用さ) 攻撃の命中、手先を使った技能の多くに関係します。10を初期値とし、2CPより上昇・下降できます。
AG(素早さ) 攻撃の回避、身体を使った技能の多くに関係します。10を初期値とし、2CPより上昇・下降できます。
VT(活力) 丈夫さ。気絶、生死の間際に関係します。10を初期値とし、1CPより上昇・下降できます。
IN(知力) 賢さ。知識、術法の修得と行使に関係します。10を初期値とし、2CPより上昇・下降できます。
WL(意志力) 意志の強さ。誘惑や恐怖の克服に関係します。10を初期値とし、1CPより上昇・下降できます。
CM(魅力) 容姿、声、仕草など。NPCへの好感度に関係します。10を初期値とし、1CPより上昇・下降できます。
HP(耐久点) 打たれ強さ。負傷(ダメージ)に耐えうる点数。ST(腕力)を初期値とし、1/2CPより上昇・下降できます。
