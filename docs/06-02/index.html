アーツ一覧

追突剣 / 易 / 刀剣・片手

盾攻撃のち通常攻撃の連続技。この技を使用したターンは盾を「止め」に使用不能。

睡魔剣 / 易 / 刀剣・片手

敵と自分とが技能勝負を行い、勝てば敵を精神的な朦朧状態にさせる。

円舞剣 / 易 / 刀剣・片手

フェイントを2回行い、次ターンの攻撃で効果の大きい方を適用できる。

失礼剣 / 並 / 刀剣・片手

敵と自分とがAGまたは「運動」勝負を行い、勝てば接近に成功し、技能値-4で即座に攻撃する。敵は「受け」と「止め」不能。

なぎ払い / 易 / 刀剣・切断

技能値-2で、屈みながら、脚を狙った攻撃を繰り出す。

切り返し / 並 / 刀剣・切断

通常攻撃を受けられたり、止められたりしたときに、即座に追撃を繰り出す。

かすみ二段 / 並 / 刀剣・切断

フェイントをかけて即座に攻撃する。

スマッシュ / 並 / 刀剣・切断

ダメージ+2、技能値-4で攻撃する。

払い抜け / 難 / 刀剣・切断・両手

敵と自分とがAGまたは「運動」勝負を行い、勝てば接近に成功し、ダメージ+2、技能値-4で即座に攻撃する。敵は「受け」不能。敵と自分との間にある程度の距離が必要。

流し斬り / 難 / 刀剣・切断・両手

成功度2以上で「受け」に成功した際、即座に反撃を繰り出す。敵は攻撃に使った武器での「受け」が不能。

二段突き / 並 / 刺突

通常攻撃を受けられたり、止められたりしたときに、即座に追撃を繰り出す。

払い突き / 並 / 刺突

フェイントをかけて即座に攻撃する。

エイミング / 難 / 刺突・両手

狙いを定めて、次ターンに射撃と同じ扱いで攻撃を繰り出す。射撃と同様、敵は「受け」が不能で、「止め」は-2の修正を受ける。

チャージ / 難 / 刺突・両手

敵と自分とがAGまたは「運動」勝負を行い、勝てば接近に成功し、ダメージ+2、技能値-4で即座に攻撃する。敵は「受け」不能。敵と自分との間にある程度の距離が必要。

急所突き / 難 / 刀剣・刺突

技能値-4で、喉を狙った一撃を見舞う。

返し突き / 難 / 刀剣・刺突

成功度2以上で「受け」に成功した際、即座に反撃を繰り出す。敵は攻撃に使った武器での「受け」が不能。

トマホーク / 易 / 殴打武器・切断・片手

武器を投げて攻撃する（射撃扱い）。

かかと切り / 易 / 殴打武器・切断・片手

技能値-2で、屈みながら、脚を狙った攻撃を繰り出す。

一人時間差 / 並 / 殴打武器

フェイントをかけて即座に攻撃する。

ハイパーハンマー / 並 / 殴打武器

ダメージ+2、技能値-4で攻撃する。

アクセルターン / 難 / 殴打武器

敵と自分とがAGまたは「運動」勝負を行い、勝てば接近に成功し、ダメージ+2、技能値-4で即座に攻撃する。敵は「受け」不能。敵と自分との間にある程度の距離が必要。

かめごうら割り / 難 / 殴打武器

盾を持った敵と自分とが技能勝負を行い、勝てば盾の破壊を試みる。盾の回避値の3倍以上のダメージが出れば破壊に成功。

脳天割り / 易 / 打撃

技能値-2で、頭を狙った一撃を見舞う。

ジャベリン / 易 / 竿状武器・刺突

武器を投げて攻撃する（射撃扱い）。

足払い / 易 / 竿状武器

技能値-2で、足首を狙った攻撃を繰り出す。振り払うので、刺突系の武器は攻撃型が「切」になる。

回転撃 / 並 / 竿状武器・打撃

通常攻撃を受けられたり、止められたりしたときに、即座に追撃を繰り出す。

はじき打ち / 並 / 竿状武器・打撃

フェイントをかけて即座に攻撃する。

風車 / 難 / 竿状武器

成功度2以上で「受け」に成功した際、即座に反撃を繰り出す。敵は攻撃に使った武器での「受け」が不能。

膝蹴り / 易 / 格闘

近接戦闘で組みついた状態で、ダメージ+1、技能値-4の攻撃を繰り出す。

回し蹴り / 易 / 格闘

近接戦闘で組みつかれていない状態で、ダメージ+2、技能値-4の攻撃を繰り出す。

浴びせ倒し / 並 / 格闘

近接戦闘で組みつかれていない状態で、フェイントをかけて即座に頭を狙った回し蹴りを見舞う。

滝登り / 並 / 格闘

近接戦闘で組みつかれていない状態で、回し蹴りを3回連続で繰り出す。

サミング / 難 / 格闘

近接戦闘で、技能値-4で、喉を狙った一撃を見舞う。

カウンター / 難 / 格闘

成功度1以上で「よけ」に成功した際、もう一度技能勝負を行い、勝てば成功度分のダメージ修正を受けて即座に攻撃を命中させる。

当て身投げ / 難 / 柔術

成功度1以上で「よけ」に成功した際、もう一度技能勝負を行い、勝てば即座に相手を投げ飛ばす。

無刀取り / 難 / 柔術

成功度1以上で「よけ」に成功した際、もう一度技能勝負を行い、勝てば即座に相手の武器を奪う。
