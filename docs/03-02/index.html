上級戦闘

姿勢・距離

姿勢

戦闘時の姿勢は、直立、屈み、膝着き、這いの4つがあり、直立以外の姿勢では、ST(腕力)、DX(器用さ)、AG(素早さ)がそれぞれ低下します。これは技能値とD-EV(よけ)に与える影響が大きいので、修正する値は、技能値とD-EV(よけ)で表記します。これは必要に応じて、能力値の低下に読み替えてください。この修正は現行ルールとも異なるので注意してください。

また敵の射手の命中判定に対しても不利な修正を与えます。直立、屈み、膝着きの姿勢時は、ターン開始時にいつでも他の姿勢に変更できます。しかし這いの姿勢時は、1ターンかけて膝着きの姿勢になることしかできません。

姿勢表
姿勢 技能値 D-EV(よけ) 射手の命中判定
直立 - - -
屈み -1 - -2
膝着き -2 -1 -4
這い -4(弩は+2) -2 -8

※竿状武器、射撃武器以外では、直立した姿勢で、転倒している敵を攻撃したり、足や足首を狙って攻撃（後述）ができない。それらを行うには、「屈み」か「膝着き」の姿勢になる必要がある(技能値-2)。

※基本ルールでは、転倒して朦朧状態から回復するまでの間、D-EVへの修正は-6だったが、上級ルールではこれは-7になる。そして自分のターンが回ってきて朦朧状態から回復し、膝着きの姿勢になったら、-2の修正になる。

距離

基本ルールでは、弓矢や呪文は命中し易く、非常に強力でした。上級ルールではこれらに、目標との距離による修正を課します。

距離による修正表
距離 修正
5m -2
10m -4
20m -6
40m -8
80m -10

※実際には、後衛から敵の前衛を目標とする場合は-2、敵の後衛を目標とする場合は-4くらいが普通。狭い屋内での戦闘なら、修正はその半分。

回避の追加ルール

空振り

敵が「よけ」に成功した場合、攻撃者は大きく空振りしてしまい、次のターンに1ターン余分な引き戻しが必要になります。当然それまでの間、武器を「受け」に使うこともできません。

軽業よけ

回避を試みるときに「軽業よけ」と宣言すると、「軽業」の技能値で「よけ」を試みることができます。このときWT(防具の重量)がペナルティとなることに注意してください。「軽業よけ」は1ターンに1度しか試みることができません。

後退

「後退」は独立した防御の方法ではなく、何らかの回避を試みるときに「後退しながら」と宣言すると、回避値に+3のボーナスを得られます。ただし1度後退すると、敵を後退させるか、自分が「全力攻撃」を仕掛けるまで、2度と後退できません。
