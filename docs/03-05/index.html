上級戦闘

近接戦闘

拳や蹴りで戦うには、敵の立ち位置にまで踏み込まなければなりません。このような格闘戦(近接戦闘)にはいくつかのルールがあります。

  • 通常の武器では直接攻撃できない。可能なのはダガーなど近接戦闘可の武器による攻撃、拳、蹴り、組み付きなど。
  • 有効な回避は「よけ」のみ。
  • 組み付かれた場合、振りほどきに成功しなければ、近接戦闘から離脱できない。

敵と自分とが技能勝負を行い、これに勝てば近接戦闘に持ち込めます。このとき片手が空いていれば、自動的に組み付くことができます。近接戦闘は比較的攻撃が当たり易いので、膠着した戦いの最中に持ち込むのも有効かも知れません。

近接戦闘での行動表
行動 説明
組み付き 敵の身体の一部をつかむ動作を組み付きと呼びます。組み付きを行うには、少なくとも片手が空いてなくてはなりません。
攻撃 拳や蹴りで攻撃します。組み付いた状態では間合いが非常に近いため、蹴りはダメージを-1した膝蹴りになります。
威力の大きい通常の蹴りを行うのであれば、その都度、技能勝負を行い近接戦闘に持ち込まなければなりません。
部位狙い ダガーやカイザーナックルなど、攻撃型が「刺し」の武器は、近接戦闘において、いずれかの部位の防具の隙間を突いて攻撃できます。部位狙いの修正が2倍になりますが、敵の防具のDR(ダメージ抵抗)が半分(端数切捨)になります。
武器を奪う 片手が空いていて、AGか「柔術」勝負に勝てば、敵の武器の柄か、敵の腕に組み付くことできます。
次のターンST勝負に勝てば、敵の武器を落とす（ダガーの場合）か、奪うことができます。このとき両手を用いている側は+2の修正があります。
近接戦闘中に落ちている武器を拾うには、AG勝負を行って敵に勝たなくてはなりません。
窒息 両手が空いていて、AGか「柔術」勝負に勝てば、敵の喉に組み付くことができます。
次のターンから自分のST(+2の修正)と、敵のVTで勝負を行い、勝った分だけダメージを与えます。
敵は窒息を振りほどいたターンは呼吸に専念することで、VT判定の成功度だけダメージが回復します。
振りほどき 組み付きを振りほどくには、自分の行動順でSTの勝負を行い、勝たなくてはなりません。両手で組み付かれている場合は敵に+2の修正があります。
振りほどきに成功したターンは、攻撃など何か行動できます。
倒し 両手が空いていて、自分のST、AG、「柔術」のいずれかと、敵のSTで勝負を行い、勝てば敵を転倒させることができます。続けて「押さえ込み」を行うなら、自分もいっしょに倒れます。もし敵も自分に組み付いているなら、このときに振りほどけます。
押さえ込み 転倒している敵に対し、「倒し」と同様の勝負を行って勝てば、敵を押さえ込んで一切抵抗できないようにしてしまいます。
