術法

効果一覧

蒼龍術

風と樹を司る系統です。

術 集中 効果
ウインドカッター n 風の刃を敵に放つ（射撃）。1集中ターンにつきダメージ1d-1(切)。
ナップ 2 敵一人を幻惑（意識はあるが積極的な行動を何もできない）状態にさせる（意志抵抗）。10ターン持続。
ダンシングリーフ 1 誰か一人に舞う木の葉のような素早さを付与する。10ターンの間、AG(素早さ)+1で、効果が累積（+3まで）。
ソーンバインド 2 茨が敵一人の足首に絡まり、足首にダメージ2d-2(切)。敵は「よけ」可能。
ミサイルガード 1 気流を操作して1ターンの間、弓矢など、攻撃型が「刺し」の射撃を、敵味方構わず全て無効にする。
風と樹の唄 1 敵対心を退け、安らぎを与える唄。1ターンの詠唱ごとに傍聴者全ての敵対反応+1で、効果が累積（+3まで）。
竜巻 3 小竜巻を起こす（範囲攻撃）。直径10mの範囲の者全てにダメージ2d-2(切)。
幻竜術 3 白い霧の竜を召喚して、戦わせる。その間、術者は戦ったり、話をしたり、他の術を使ったりできない。
白竜は毎ターン、ダメージ2d-2(切)の風の刃を飛ばすことができる。

朱鳥術

炎と生命を司る系統です。

術 集中 効果
ファイアボール n 小さな火球を敵に放つ（射撃）。1集中ターンにつきダメージ1d(叩)。
ハードファイア 1 誰か一人の戦意を向上させる。10ターンの間、戦闘技能値+1で、効果が累積（+3まで）。
セルフバーニング 2 術者を炎の膜で包む。10ターンの間、直接攻撃に失敗した敵に反撃し、2d(叩)のダメージ。
術がかかっている間、熱や冷気による攻撃のダメージを無効にする。また一度反撃すると術が解ける。
生命の息吹 2 気絶している者を覚醒させる。HP(耐久点)が-最大値まで達している場合は無効。
ファイアウォール 1 炎の壁を起こす。1ターンの間、味方全員へのセルフバーニングの効果。
魂の歌 1 戦意を向上させる歌。1ターンの詠唱ごとに味方全員へのハードファイアの効果。
（ハードファイア、魂の歌の効果は、足し合わせて+3まで累積。）
イニシレイション 3 大きな火球を放つ（射撃）。ダメージ4d(叩)。「止め」に用いた盾は炎上する。
リヴァイヴァ 3 気絶または死亡している者のHP(耐久点)を完全回復し、蘇生させる。死後1時間経過しているごとに-1の修正。

白虎術

石と獣を司る系統です。

術 集中 効果
ストーンバレット n その辺に転がっている石粒などを敵に放つ（射撃）。1集中ターンにつき一石を投じ、ダメージ1d+1(叩)。
ベルセルク 1 誰か一人のST(腕力)とVT(活力)を+1するが、意志抵抗に失敗すると、狂戦士化。10ターン持続し、効果が累積（+3まで）。
カムフラージュ 1 誰か一人の姿を消す。積極的な行動をとると意志判定-4を行い、失敗すると術が解けてしまう。
アースハンド 2 敵一人の脚を、大地に鷲づかみされたかのように、動けなくする。10ターンの間、足の故障と同じ（意志抵抗）。
ストーンスキン 0 瞬間的に自分自身の皮膚を堅くする。白虎術の技能値で、「止め」を行うのと同じ。1ターンに一度だけ。この術を使うと、それまでの精神集中は無効になる。
タッチゴールド 0 近接戦闘中、触れた敵にこの術を使うと、敵は金塊になる（意志抵抗）。術者がもう一度触れると元に戻る。
クラック 3 敵一人の足元に地割れを起こし、戦闘から離脱させる。敵は「よけ」可能。
戦闘後は誰かの手助けによって、這い上がって出ることができる。
光の壁 1 味方全員を守る光の壁を作り出す。1ターンの間、味方全員のDR(ダメージ抵抗)+3。

玄武術

水と時を司る系統です。

術 集中 効果
生命の水 n 誰か一人のHP(耐久点)を、1ターンの集中につき最大値の3分の1（端数切り上げ）回復する。
3ターンの集中で、身体の一時異常も同時に治す。
神秘の水 2 誰か一人の身体と精神の一時異状を治す。
ウォーターポール 1 誰か一人に水の鎧を作る。10ターンの間、DR(ダメージ抵抗)+1で、効果が累積（+3まで）。
スパークリングミスト 2 濃霧を発生させる。敵味方ともに距離による修正が倍になる。10分ほど持続する。
スコール 3 豪雨を降らせる。敵味方ともに距離による修正が倍になり、炎のダメージが半減する。10分ほど持続する。
召雷 3 雷を起こす（射撃）。ダメージ3d-3(叩)だが、金属性の装備を身につけているとDRは0、回避判定-2。
時間遡行 3 時の流れを遡行し、直前の誰かの行動を無効にする。この術は3ターン目の集中を開始していれば、いつでも発動できる。
クイックタイム 3 時の流れを歪める。1ターン、PC（術者と親しい者数名）のみが行動可能。

陽術

陽光と空間を司る系統です。

術 集中 効果
サンシャイン 2 眩しい閃光を放つ（範囲攻撃）。見た者全員の技能値-6、D-EV(よけ)-2。
次ターン以降、ターン開始時にVT(活力)によるD-EV(よけ)の回復判定を行う。技能値の修正は必ず1ターン持続。
ヒートウェイブ 2 熱風を起こす（範囲攻撃）。直径10mの範囲の者全てにダメージ2d-2(叩)。
スターフィクサー 2 敵一人の腕を痺れさせる。腕にDR無視のダメージ2d-2。意志抵抗によりダメージ半減。
デイブレイク 2 誰か一人を瞬間移動させ、戦闘から離脱させる。不利な修正無しで、意志抵抗により効果消滅。
デイブレイクで瞬間移動させられる場所は、術者の記憶にある場所のうちの何処か。PCの場合、戦闘後すぐに再会できる。
光の剣 1 術者の手にソード+2(最上質)と同等の性能の光の剣を作り出す。
光の剣は、術者の意思で伸縮自在で、ダガー、バスタードソードの性能でも扱うことができる。
光の剣は決して落とすことはなく、また他人に譲渡はできない。
太陽風 3 太陽風を起こす（範囲攻撃）。直径10mの範囲の者全てにダメージ3d-3(叩)。
再生光 3 誰か一人のHP(耐久点)を完全回復し、身体の一時異常と、部位の故障も治す。
幻日 3 術者の分身を作り出す。分身によって、1ターンに同じ行動を2ターン分行える。10ターン持続。

陰術

月光と精神を司る系統です。

術 集中 効果
ムーンシャイン n 誰か一人のHP(耐久点)を、1ターンの集中につき最大値の3分の1（端数切り上げ）回復する。
3ターンの集中で、精神の一時異常も同時に治す。
ムーングロウ 1 誰か一人の精神力を高める。10ターンの間、WL(意志力+1で、効果が累積（+3まで）。
ソウルフリーズ 2 敵を恐怖で凍りつかせる。敵はWL(意志力)-2で恐怖判定を行う。
サクション n 敵のHP(耐久点)を奪う。1集中ターンにつき1d-2。意志抵抗によりダメージ半減。アンデッドには逆効果。
月読の鐘 1 気絶している者を意のままに操る。1ターンの集中につき、1ターン分の命令を下せる。
幻惑光 3 敵一人に、1dの出目に従い、以下のような狂気を起こす（意志抵抗）。10ターン持続。
1.幻惑/宙をみつめ、ぼーっとする。周囲の出来事にまるで気が付かない。傷を負ったりすると我に返る。（蒼龍術「ナップ」と同じ。）
2.狂戦士/最も近くにいる者に、ひたすら全力攻撃を繰り返す。射撃武器は、狙いをつけずに矢が無くなるまで乱射する。
3.恐怖/恐怖で凍りつく。WL(意志力)-4で恐怖判定を行う。
4.混乱/げらげら笑ったり、冗談を飛ばしたり、くすくすほくそ笑んだりする。危険に直面すると、50%の確率で愚かな行動をとる。
5.魅了/術者の命令に何でも従う。命令が下されなくても、術者が一番喜びそうな行動をとる。ただし危険を伴う行為はしない。
6.忠実/魅了とほとんど同じだが、危険を顧みずに術者の命令に従い、術者が一番喜びそうな行動をとる。
アンティマジック 3 誰か一人に働いている、持続時間のある術の効果を打ち消す。
シャドウサーバント 3 誰か一人の分身を作り出す。「幻日」とほとんど同じだが、他者に本人と分身の判別が可能。
