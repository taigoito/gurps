基本戦闘

ダメージ効果

朦朧

一度にST(体力)の3分の1ないし半分以上のダメージを受けると、朦朧状態に陥ります。前者は軽い朦朧状態、後者は重い朦朧状態と呼びます。

この状態では、ターン開始時の回復判定(ST判定)に成功するまで一切行動できない上、回避も試みることができません。重い朦朧状態の場合は、回復判定に-4の修正を受けます。

転倒

一度にST(体力)の半分以上のダメージを受けると、転倒判定(-2の修正でST判定)を行います(成功しても朦朧状態です)。

転倒したら、起き上がったターンの終わりまで、D-EV(よけ)に-4の修正を受けます。

気絶と死亡

一度にST(体力)以上のダメージを受けると、気絶します。

さらにこのとき、ダメージがSTを超えた分をマイナスの修正としてWL(意志力)判定を行い、これに失敗すると、人物は死亡します。いわゆるゲームオーバーですので、GMと相談してどこかのタイミングで新しいPCを作成し、再度合流を図ってください。

このルールだと厳しすぎると判断する場合、上級ルールの部位の「故障の期間」を参考にしてください。
