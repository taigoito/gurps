キャラクター

装備

装備一覧から技能に応じた好きな武器、防具を選び、必要な数値を書き写して下さい。必要腕力のような概念はありません。

武器
値 説明
Dmg(ダメージ) 攻撃力を表します。dはダイスの数です。武器の基本ダメージに、本人の腕力によるダメージを足した値です。
LV(技能値) 命中率を表します。
P-EV(受け) 「受け」を試みるとき、武器によって有利に補正される回避率を表します。
防具
値 説明
DR(ダメージ抵抗) 防御力を表します。
B-EV(止め) 「止め」を試みるとき、盾によって有利に補正される回避率を表します。
WT(重量) 「よけ」を試みるとき、重量によって不利に補正される回避率を表します。
