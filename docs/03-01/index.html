上級戦闘

攻撃型

上級戦闘では、武器を用途によって「叩き」、「切り」、「刺し」の3タイプ(攻撃型)に分けます。鋭利な刃を持つ武器は、基本ダメージ(衝撃)がいくらか減じるかわりに、衝撃に比例した、裂傷によるダメージ変化が加わります。

  • 「叩き」は、肉体を打ち砕く攻撃型で、防具性能を抜けたダメージはそのままです。
  • 「切り」は、肉体を切り裂く攻撃型で、防具性能を抜けたダメージを1.5倍にします。
  • 「刺し」は、肉体を突き刺す攻撃型で、防具性能を抜けたダメージを2倍にします。

武器と、本人の筋力によるダメージを合わせ、ダメージダイスを修正する値（例えば1d+2なら+2の部分）が+3以上をとる場合、+3を1d-1に置き換えます。例えばST(筋力)12でソードを振るうと、ダメージは1d+3になりますが、+3を1d-1に置き換え、2d-1にします。
